iview表格中icon图标鼠标移动上去显示文字
css iview
Vue
发布日期:   2019-09-13
文章字数:   176
阅读时长:   1 分
阅读次数:  

需求

在table表格中,操作按钮一栏默认显示小眼睛图标.当鼠标移动上去的时候icon小眼睛隐藏,”查看详情文字”显示

解决方案

渲染表头时添加class名,通过css控制隐藏显示

    {
          title:'操作',
          key:'action',
          className:'hover_text',
          render:(h,params) => {
            return h('div',[
              h('Icon',{
                class:'look_eye',
                props:{
                  type:'md-eye',
                  size:'20',
                  color:'#FFAA06'
                }
              }),
              h('strong',{
                class:'look_text',
                style:{
                  color:'#FFB72C',
                  display:'none'
                },
                on:{
                  click:()=>{

                  }
                }
              },'查看详情')
            ])
          }
        }

css

.hover_text{
  cursor: pointer;
}
.hover_text:hover .ivu-icon{
  display: none;
}
.hover_text:hover .look_text{
  display: block !important;
}
