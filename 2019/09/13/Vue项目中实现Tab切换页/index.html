前言

刚刚开始上手Vue项目就遇到一个小问题,tab切换页面不知道怎么写.后来同事给了个思路,使用v-for来写

实现目标


如图,有三个tab,默认情况下标题的字和icon都是蓝色,选中的时候字和icon都要变成白色.同时选中项右边出现一个小三角(最后一个除外)

开始动手

  1. 写一个div
    a 通过.tab_btn默认给所有button蓝色样式
    b 通过判断index是否=active动态绑定class名”active”
    c icon的加载哪张图片取决于index是否等于active
    <div class="header">
        <Row class="home_tab_title">
            <Col span="6" class="header_col" v-for="(item,index) in btnList" :key="index">
              <button class="tab_btn" :class="{'active': index == active}"  @click="handleBtn(index)">
                <img :src="index == active ? item.link : item.src" alt="" class="btn_img">
                {{item.btnName}}
              </button>
              <div class="triangle_right_blue" v-show="index == active && index != btnList.length-1" style="top:15px;right:-30px;z-index:5;"></div>
            </Col>
        </Row>
    </div>

  1. 写我们的方法事件
    a 写一个数组,渲染tab标题和里面的icon
    b icon给了两个地址,一个是蓝色图标,一个是白色图标
    c 默认active=0 初始化是选中第一个tab

     btnList:[
     {
       src:require('../assets/img/dna.png'),
       link:require('../assets/img/light_dna.png'),
       btnName:'DNA提取'
     },
     {
       src:require('../assets/img/kuozeng.png'),
       link:require('../assets/img/light_kuozeng.png'),
       btnName:'PCR扩增'
     },
     {
       src:require('../assets/img/dianyong.png'),
       link:require('../assets/img/light_dianyong.png'),
       btnName:'电泳'
     },
     {
       src:require('../assets/img/result.png'),
       link:require('../assets/img/light_result.png'),
       btnName:'结果管理'
     }
   ],
   active:0,

    handleBtn(index){
       this.active = index
     },

    总结

    如此,即可实现一个简单的tab切换效果~
    我觉得使用v-for来写可以动态的控制加载项,v-for中的index可以反馈给我们点击的是第几个tab.小三角形最后一个不显示通过index != btnList.length-1
